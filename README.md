
# Expense Analysis Project

## Overview
This project analyzes 6 months of personal expense data using Microsoft Excel and PowerPoint.
The goal is to identify spending patterns, major expense categories, monthly trends, and provide savings recommendations.

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Data Analysis)
- Microsoft PowerPoint (Data Visualization & Presentation)

## Key Analysis
- Month-wise expense trends
- Category-wise expense distribution
- Top 2 expense categories comparison
- Item-wise monthly spending analysis
- Savings insights and recommendations

## Files
- Expense details for 6 months (7).xlsx → Raw dataset and analysis
- Expense-Analysis-Presentation.pptx → Final presentation

## Skills Demonstrated
- Data cleaning and organization
- Pivot table analysis
- Chart creation and interpretation
